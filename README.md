# Face Attendance System

## Overview
A Face Recognition Based Attendance System developed using Python, OpenCV, and Machine Learning.

## Features
- Face Registration
- Face Training
- Real-Time Face Recognition
- Automatic Attendance Marking
- Attendance CSV Export

## Technologies Used
- Python
- OpenCV
- NumPy
- Pandas

## Project Workflow
1. Capture Face Images
2. Train Face Recognition Model
3. Recognize Faces
4. Mark Attendance Automatically
